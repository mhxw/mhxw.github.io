在Ubuntu上安装Golang

Posted on 2019-02-26 In 技术教程

下载

https://golang.org/dl/
wget -c https://golang.org/dl/go1.16.5.linux-amd64.tar.gz -O - | sudo tar -xz -C /mhxw

设置环境变量

vim ~/.profile
export PATH=$PATH:/mhxw/go/bin
source ~/.profile

校验版本

go version
