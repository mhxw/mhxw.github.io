腾讯云ubuntu系统切换到root权限

Posted on 2020-08-23 In 技术教程
  1. 生成root用户

设置root的密码，建议和ubuntu的密码一样

script
sudo passwd root
  1. 编辑sshd_config文件
  • 编辑以下文件
script
sudo vi /etc/ssh/sshd_config

  • 找到PermitRootLogin，重新复制一行然后将其后面的文本改为yes

  • 保存并退出

  • 重新启动ssh服务

script
sudo service ssh restart
  1. 重新登录

使用root用户名登录ssh，操作完成
