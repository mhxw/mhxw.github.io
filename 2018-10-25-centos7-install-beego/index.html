Centos7.5安装部署Golang、Beego框架

Posted on 2018-10-25 In 技术教程

安装Golang

  1. 进入home目录下
    cd /home
  2. mkdir Go 创建一个Golang的目录
    mkdir Go
cd Go
  3. 在 http://www.studygolang.com/dl 找到go对应的安装包

  1. 复制下载文件链接，在命令行执行代码：

    wget https://dl.google.com/go/go1.11.linux-amd64.tar.gz

  2. 解压缩

    tar xzvf  go1.11.linux-amd64.tar.gz

  3. 设置环境变量：

    vi ~/.bashrc

  4. 添加：

    export GOROOT=/home/Go/go
export PATH=$GOROOT/bin:$PATH
export GOPATH=/home/Go/go-project
export PATH=$PATH:$GOPATH/bin

8、填完执行：

source ~/.bashrc

安装Beego

  1. 安装git
    yum install git

  1. git下载beego和bee

    go get -u -v [http://github.com/astaxie/beego](http://github.com/astaxie/beego)
go get -u -v githubcom/beego/bee

    下载时等待一会儿，直到结束为止

    此时，Go和beego就安装好了

  2. 测试：

    cd $GOPATH/src
bee new class

    successfully，测试ok~

  3. 进入class目录

    cd class

  4. 运行mian.go文件

    go run main.go

  5. 打开8080访问，显示欢迎页面，ok~
