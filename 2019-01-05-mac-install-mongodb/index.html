Mac安装配置MongoDB

Posted on 2019-01-05 In 技术教程

安装MongoDB

  1. 打开官方下载网站
script
https://www.mongodb.com/download-center/community


2. 进入/usr/local目录

script
cd /usr/local
  1. 下载并解压
script
sudo curl -O https://fastdl.mongodb.org/osx/mongodb-osx-ssl-x86_64-4.0.9.tgz
sudo tar -zxvf mongodb-osx-ssl-x86_64-4.0.9.tgz
  1. 重命名为 mongodb 目录
script
sudo mv mongodb-osx-x86_64-4.0.9/ mongodb
  1. 把mangodb的二进制命令目录添加到PATH路径中
script
export PATH=/usr/local/mongodb/bin:$PATH

运行MongoDB

  1. 首先创建一个数据库存储目录 /data/db
script
sudo mkdir -p /data/db
  1. 启动 mongodb，默认数据库目录则为 /data/db
script
sudo mongod

  1. 再打开一个终端，执行以下命令，进入MangoDB
script
cd /usr/local/mongodb/bin
./mongo

下载MongoDB-Compass

  • 进入官网直接下载
script
https://www.mongodb.com/download-center/compass
  • 或者执行以下命令下载
script
sudo curl -O https://downloads.mongodb.com/compass/mongodb-compass-1.18.0-darwin-x64.dmg
  • 双击安装运行即可

FAQ

  • 如果你的数据库目录不是/data/db，可以通过 –dbpath 来指定。
script
sudo mongod --dbpath=/data/db
# mac # mongodb # database
